# Luvascript Datatypes

## primitive datatypes

| Type | Size |
|------|------|
| `i8` | 1 byte |
| `i16` | 2 bytes |
| `i32` | 4 bytes |
| `i64` | 8 bytes |
| `u8` | 1 byte |
| `u16` | 2 bytes |
| `u32` | 4 bytes |
| `u64` | 8 bytes |
| `f32` | 4 bytes |
| `f64` | 8 bytes |
| `bool` | 1 byte |
| `string` | variable |
