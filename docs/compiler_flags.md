# Luva Compiler Flags

-   `-f [format]`		format - specify file format
-   `-o [filename]`		specify output filename
-   `-c`				disable linking
-	`-S`				generate assembly code
-   `-asm:[plattform]` 	enable inline assembly ([See more](flags/inline-assembly.md))

    Supported Plattforms
    -   `x86-64`
    -   `arm64`
