# Luvascript Roadmap

### New Features

- [ ] classes
- [ ] interfaces

## Luvascript v0.2.0

### New Features

- [ ] more primitive datatypes

- [ ] arrays

- [x] basic mathematical expressions

- [x] conditional statements (`if` / `else if` / `else`)
- [x] condtional expressions

- [x] jump statememts (`return`)
- [x] compound statements
